# DESIGN.md

## 气质与意象

**政务数据中心控制台** — 省级大数据局不动产抵质押品核对系统，整体气质应当传达"权威、精确、可信赖"。视觉锚点：深冬清晨的政府办公楼 — 大面积灰白墙面、深蓝玻璃幕墙、整齐划一的档案柜、冷白日光灯下的数据屏幕。一切克制有序，信息密度高但不混乱。

## 配色方案

| 角色 | 色值 | 意象来源 |
|------|------|----------|
| 主色（Primary） | `#1e3a5f` 深海蓝 | 政府机关深蓝标识、档案柜金属色 |
| 主色浅 | `#2d5a8e` | 玻璃幕墙映出的天光 |
| 辅助色 | `#f0f4f8` 浅灰蓝 | 办公楼墙面、打印纸 |
| 强调色（Accent） | `#d4a017` 赭金 | 公章印泥、政府文件金边 |
| 成功 | `#16a34a` 绿 | 核对一致 |
| 警告 | `#ea580c` 橙 | 待核实/变更 |
| 危险 | `#dc2626` 红 | 核对不一致/注销 |
| 背景色 | `#f8fafc` | 档案室灯光下的白墙 |
| 卡片背景 | `#ffffff` | 打印纸 |

## 字体排版

- 字体族：系统字体栈 `-apple-system, BlinkMacSystemFont, "PingFang SC", "Microsoft YaHei", sans-serif`
- 标题：font-weight 600，深蓝色
- 数据标签：font-size 12px，颜色 `#64748b`
- 数据值：font-weight 500，深色
- 表格行高紧凑，行间距适中

## 视觉策略

- 图形语言：直线、网格、数据密度优先
- 卡片边框：1px solid `#e2e8f0`，圆角 8px
- 表格：斑马纹（隔行浅灰底 `#f8fafc`），表头深蓝底白字
- 状态标签：小圆角 pill badge，语义色彩填充

## 页面结构

### 侧边栏导航
- 深蓝底色 `#1e3a5f`
- 白色文字 + 图标
- 当前项高亮：赭金左边框 + 浅蓝底

### 顶部状态栏
- 白色背景，底部细线
- 左侧面包屑，右侧用户/通知

### 内容区
- 统计卡片行：4 个关键指标卡
- 主操作区：查询表单 + 数据表格
- 核对面板：左右对照或上下对照

## 动效与交互

- 过渡：150ms ease，克制不张扬
- 悬停：行高亮（浅蓝底），按钮微亮
- 状态切换：淡入淡出 200ms
- 加载：骨架屏，不用 spinner
- 核对动画：数据行逐一扫描效果

## 设计禁忌

- 禁止渐变背景（政务系统不用渐变）
- 禁止卡通图标或插图
- 禁止大圆角（>12px）
- 禁止鲜艳色彩大面积使用
- 禁止花哨的动画效果
- 禁止浅色文字在浅色背景上
